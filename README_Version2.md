# أكاديمية — نسخة Dark + Liquid Glass (Static)

هذا المشروع هو نسخة ثابتة (Static) لموقع أكاديمية بتصميم "Dynamic Black" وتأثيرات "Liquid Glass" جاهزة للرفع على GitHub Pages.

الملفات:
- index.html — الصفحة الرئيسية بواجهة زجاجية ديناميكية.
- dashboard.html — لوحة الطالب.
- styles.css — أنماط التصميم (Dark + Glassmorphism + Animated blobs).
- data.js — بيانات الدورات المشتركة.
- script.js — منطق العرض، المحاكاة الشرائية، وlocalStorage.

كيفية الاستخدام السريع:
1. أنشئي مستودعًا جديدًا على GitHub.
2. ارفعي كل الملفات أعلاه (Upload files) أو قومي بـ git push.
3. اذهبي إلى Settings > Pages واختر الفرع `main` أو `gh-pages` واضغطي Save.
4. انتظري قليلًا، ثم زوري العنوان user.github.io/repo-name

ملاحظات:
- كل الوظائف الحالية تعمل محليًا (localStorage). الدفع محاكاة فقط.
- لتوصيل بوابة دفع حقيقية أو عمل Backend (مزامنة مستخدمين، حفظ تقدم على الخادم، حماية الفيديو)، أستطيع إعداد Backend بسيط (Node/Express أو Firebase) وتوصيله لاحقًا.
- إذا تريدين أرفع الملفات مباشرة في مستودع باسمك، أعطني اسم المستودع (مثال: Dou2m/academy) وسأقوم بالرفع أو أعرض لك تعليمات git خطوة بخطوة.
